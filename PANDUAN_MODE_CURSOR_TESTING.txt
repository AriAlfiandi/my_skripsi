================================================================================
üìä JENIS PENGUJIAN KONTROL KURSOR
================================================================================

1. PRESISI (Precision Test)
   - 9 titik target di layar (grid 3x3)
   - Ukur deviation dari target
   - Success rate jika deviation < 50px
   - Output: Avg Deviation, Min, Max

2. SMOOTHNESS (Kehalusan Tracking)
   - Tracking pergerakan kursor
   - Hitung jitter (variasi gerak)
   - Smoothness score (0-100)
   - Output: Jitter (px), Smoothness Score

3. KECEPATAN (Speed Test)
   - Test gerakan lambat
   - Test gerakan normal
   - Test gerakan cepat
   - Output: Response time per kategori

üéÆ CARA MENGGUNAKAN
================================================================================

STEP 1: JALANKAN PROGRAM
-------------------------
python virtual_mouse_cursor_testing.py

STEP 2: AKTIFKAN MODE KURSOR
-----------------------------
Tekan 'C' untuk masuk ke Mode Testing Kursor

Yang berubah:
‚úì 9 titik target muncul di layar
‚úì Info mode di layar: "MODE KURSOR: ON"
‚úì Real-time stats: Jitter, Smoothness, Presisi

STEP 3: LAKUKAN PENGUJIAN
--------------------------

A. TEST PRESISI (9 Titik)
~~~~~~~~~~~~~~~~~~~~~~~~~~
1. Mode Kursor: ON (tekan C)
2. Lihat target points di layar (angka 1-9)
3. Target aktif berwarna HIJAU
4. Arahkan jari tengah ke target
5. Tekan 'P' untuk record hit
6. Ulangi untuk semua 9 target
7. Lihat hasil di terminal

Contoh Output Terminal:
  üéØ Target 1/9: Deviasi 15.32px
  üéØ Target 2/9: Deviasi 22.45px
  ...
  ‚úÖ Test Presisi Selesai!
     Avg Deviation: 18.75px
     Success Rate: 88.89%

B. TEST SMOOTHNESS
~~~~~~~~~~~~~~~~~~
1. Mode Kursor: ON
2. Tekan 'M' untuk mulai tracking
3. Gerakkan jari tengah membentuk pola (lingkaran/zigzag)
4. Lihat cursor trail (garis kuning)
5. Smoothness auto-tracked
6. Lihat stats di layar: "Jitter: 5.2px | Smooth: 94.8"

C. TEST KECEPATAN
~~~~~~~~~~~~~~~~~
1. Mode Kursor: ON
2. Untuk gerakan LAMBAT:
   - Tekan '1' (mulai timer)
   - Gerakkan jari tengah perlahan
   - Tekan '1' lagi (stop & record)
   
3. Untuk gerakan NORMAL:
   - Tekan '2' (mulai timer)
   - Gerakkan jari tengah normal
   - Tekan '2' lagi (stop & record)
   
4. Untuk gerakan CEPAT:
   - Tekan '3' (mulai timer)
   - Gerakkan jari tengah cepat
   - Tekan '3' lagi (stop & record)

STEP 4: LIHAT STATISTIK

Tekan 'S' untuk lihat statistik lengkap:
